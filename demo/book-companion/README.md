## Try it yourself

```bash
git clone https://github.com/mduongvandinh/llm-wiki.git my-dune-wiki
cd my-dune-wiki

# Set up book-companion variant
/llm-wiki setup book-companion

# Drop your own notes into raw/articles/ then:
/llm-wiki ingest
/llm-wiki book-summary
```

The demo used Dune (Frank Herbert) as sample data. Works equally well for any book — fantasy, sci-fi, literary fiction, non-fiction.
